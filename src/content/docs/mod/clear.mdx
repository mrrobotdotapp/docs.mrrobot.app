La commande `/clear` permet de supprimer un certain nombre de messages dans un salon.
Cette commande est réservée aux modérateurs du serveur.

## Utilisation

```
/clear <nombre>
```

:::note
Le nombre de messages à supprimer doit être compris entre 2 et 100. La limitation est imposée par Discord.
:::

## Permission

Permission par défaut : **Gérer les messages**.
