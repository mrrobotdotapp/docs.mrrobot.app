##  Soucis avec le système contre les insultes

- Vérifier si le robot a accès au salon textuel.
- ~~Si vous avez les droits de supprimer les messages, alors vous n'êtes pas affecté par ce filtre.~~

- [Inviter de nouveau le robot sur votre serveur](https://mrrobot.app/add), car il pourrait lui manquer des permissions.

## Puis-je ajouter dans la liste par défaut une insulte ?

Oui ! Vous pouvez faire votre proposition par email ou directement sur le serveur Discord.

## Comment ajouter le robot sur mon serveur Discord ?

Il vous suffit de vous rendre sur la page [d'invitation du robot](https://mrrobot.app/add) et de suivre les instructions.

## Quelle est la différence entre la version gratuite et la version premium ?

La version gratuite est limitée en fonctionnalités, tandis que la version premium offre des fonctionnalités supplémentaires telles que **la modération par IA**.
Pour une liste complète des différences, veuillez consulter la page [Premium](https://mrrobot.app/premium).

## Dois-je faire faire une commande pour activer la modération ?

Non. C'est le principe du robot, dès que vous l'ajoutez sur votre serveur Discord, les paramètres du robot font que le système contre les insultes est activé par défaut !
Si vous désirez faire l'inverse, c'est-à-dire de le désactiver, veuillez suivre [la documentation pour configurer le filtre](/configuration/swear/config-filter/).

## Comment Mr. Robøt peut-il améliorer la sécurité de mon serveur Discord ?

Mr. Robøt utilise une intelligence artificielle avancée pour détecter et modérer les comportements inappropriés sur votre serveur Discord.
Cela permet de maintenir un environnement sûr et convivial pour tous les membres de votre communauté.

## Comment configurer les rôles et salons pour limiter l’impact des insultes ?

Vous pouvez configurer les rôles et salons en utilisant les commandes de configuration disponibles dans Mr. Robøt.
Par exemple, vous pouvez définir des rôles spécifiques qui ont des permissions différentes en matière de modération, ou configurer des salons où certaines règles s'appliquent.
Pour plus de détails, consultez la section [Configuration des salons](/configuration/swear/config-channels/).

## C'est quoi les Slash Commands ?

Les *Slash Commands* sont des commandes qui commencent par `/` et qui permettent d'interagir avec le robot.
C'est une nouvelle fonctionnalité de Discord qui permet de simplifier l'utilisation des robots.
