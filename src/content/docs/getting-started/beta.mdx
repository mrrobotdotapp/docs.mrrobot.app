description: Les informations pour accéder à la bêta du robot Discord.
---

## Qu'est-ce qu'une bêta ?

Une bêta est une version d'une fonctionnalité et/ou une commande qui n'est pas encore terminé. Elle est donc susceptible de contenir des bugs et des problèmes. C'est pour cela que nous vous demandons de nous remonter les problèmes que vous rencontrez avec le robot.

## Comment y accéder ?

Pour accéder à la bêta, il vous suffit de faire la commande `/beta` sur **votre propre serveur Discord** dont vous avez la permission nécessaire pour exécuter la commande. Vous aurez alors accès à la bêta du robot.


![Preview de la commande /beta](/img/beta.png)

:::note
Par défaut, la commande n'est executable que par les administrateurs du serveur. Vous pouvez modifier les permissions directement dans vos paramètres de serveur.
:::

## Comment sait-on que l'on est en bêta ?

En exécutant la commande `/beta`, vous recevrez un message vous indiquant que vous êtes en bêta. Dans le cas contraire, vous serez donc en version stable.

## Comment savoir si une fonctionnalité ou une commande est en bêta ?

Deux moyens s'offrent à vous pour savoir si une fonctionnalité ou une commande est en bêta :
- Vous avez la liste des fonctionnalités et des commandes en bêta avec la même commande `/beta`
- Quand vous exécutez une commande en bêta, vous y verrez un badge `BETA` à côté du nom de la commande ainsi qu'un message vous indiquant que vous êtes en bêta.


## La commande n'est pas disponible

Il se peut que la commande ne soit pas disponible sur votre serveur. Cela peut être dû à plusieurs raisons :
- Vous n'avez pas les permissions nécessaires pour exécuter la commande
- Le robot n'est pas sur votre serveur
- Le robot est en maintenance
- La commande est désactivée sur votre serveur
- Le robot est en bêta fermée
