Vous avez la possibilité **d'activer et de désactiver** le filtre contre les insultes sur **l'entièreté de votre serveur** avec la commande adéquate.

```
/paramètres-insultes Activer/désactiver le filtre contre les insultes
```

Même si le filtre est désactivé, vous pouvez toujours configurer les salons qui ne seront pas concernés par celui-ci, ainsi que les rôles.
Vous avez la liste des insultes affichées dans la [FAQ](/getting-started/faq/#puis-je-avoir-la-liste-des-insultes).

:::note
Le filtre contre les insultes est activé par défaut.
:::

## Permission

Permission par défaut : **Gérer le serveur**.
