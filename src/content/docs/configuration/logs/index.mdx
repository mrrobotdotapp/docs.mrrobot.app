## Présentation

Mr. Robøt vous permet de suivre, enregistrer et gérer les activités de votre serveur Discord facilement.
Les logs sont des messages envoyés par le robot pour vous informer des évènements importants sur votre serveur Discord.
Vous pouvez choisir les évènements à logger et le salon textuel où les logs seront envoyés.

## Configuration
<Steps>
1. Ajouter [le robot sur votre serveur Discord](https://mrrobot.app/add)

2. Exécuter la commande
    ```
    /paramètres-logs
    ```

3. Un salon de logs (webhook) est créé automatiquement si aucun n'existe encore, puis choisir les évènements à logger dans le menu déroulant du panneau affiché
    ![Message de Mr. Robøt pour configurer les évènements à logger](/img/logs-add-remove-events.png)

4. Tester si vous le souhaitez la connexion en cliquant sur le bouton **Tester le webhook** du même panneau
</Steps>

## Déplacer le salon des logs

<Steps>
  1. Pour pouvoir déplacer le salon des logs, vous devez simplement modifier le salon textuel choisi en allant dans **Paramètre du serveur > Intégrations > Mr. Robøt > Webhooks**.
  2. Sélectionnez le salon où vous souhaitez que les logs soient envoyés.
     ![Déplacer le salon des logs](/img/logs_edit_webhook.png)
  3. Cliquez sur **Enregistrer les modifications**.
</Steps>


## J'ai supprimé le salon des logs

Oh... ça peut arriver, pas de panique ! Pour régénérer le webhook, suivez les étapes suivantes :

<Steps>
  1. Exécutez la commande
      ```
      /paramètres-logs
      ```
  2. Vous devriez voir un message vous indiquant le nouveau salon des logs et une aide, ainsi qu'un message qui s'envoie dans le salon des logs.
     ![Message du robot Discord indiquant les étapes à suivre pour configurer salon des logs](/img/logs_generate_webhook_first_time.png)
  3. Vous pouvez également tester la connexion du webhook en cliquant sur le bouton **Tester le webhook** du panneau affiché
  4. Tout est bon, vous avez régénéré le webhook avec succès ! 🎉
</Steps>

## Résolution des problèmes

### Aucun évènement n'est loggé

Veuillez vérifiez :
- Que le robot a bien [les permissions nécessaires](#la-permission-gérer-les-webhooks)
- Que le robot est bien connecté à votre serveur Discord
- Que vous avez bien suivi les instructions du robot
- Que vous avez bien choisi les évènements à logger, vous pouvez voir la liste des évènements disponibles avec la commande `/paramètres-logs`
- Que le salon textuel choisi pour les logs est toujours présent, si ce n'est plus le cas, régénérez le webhook avec la commande `/paramètres-logs`

## La permission Gérer les webhooks

<Steps>
  1. Pour donner la permission à Mr. Robøt, vous devez avoir un rôle adéquat pour modifier les permissions demandées.
  2. Allez dans **Paramètres du serveur > Rôles > Mr. Robøt**.
  3. Recherchez "webhook" et cochez les permissions nécessaires.
     ![Donner la permission "Gérer les webhooks" à Mr. Robøt](/img/logs/activateWebhookPermission.gif)
  4. Cliquez sur **Enregistrer les modifications**.
  5. Vous avez donné la permission à Mr. Robøt pour gérer les webhooks avec succès ! 🎉
</Steps>

## Permission

Permission par défaut : **Gérer le serveur**.
