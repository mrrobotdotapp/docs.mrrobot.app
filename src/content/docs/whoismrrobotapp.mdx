:::danger[Service arrêté]
**[whois.mrrobot.app](https://whois.mrrobot.app) est déprécié et n'est plus disponible.** Le service a été arrêté définitivement. Utilisez la commande `/whois` de Mr. Robøt directement dans Discord à la place.
:::

## Qu'était whois.mrrobot.app ?

**whois.mrrobot.app** était un site web open source permettant d'obtenir des informations sur un utilisateur ou robot Discord à partir de son identifiant, sans avoir besoin d'être connecté à Discord.

### Fonctionnalités

- Informations complètes : identifiant, nom d'utilisateur, discriminateur, date de création du compte, nombre de jours depuis la création.
- Si la cible est un robot, possibilité d'ajouter directement le robot depuis la page.
- Compatible **PWA & TWA** — transformable en application Android.
- Affichage du profil Discord complet.
- Bonnes performances et accessibilité.

## Pourquoi ce service est-il arrêté ?

Vous pouvez toujours consulter les informations d'un utilisateur Discord directement via la commande `/whois` de Mr. Robøt, sans quitter Discord.

Par souci de transparence, whois.mrrobot.app a également été arrêté pour rester en conformité avec les **conditions d'utilisation de l'API Discord**. Ce type d'outil permettait de rechercher des profils sans serveur commun, ce qui contourne les règles de visibilité de Discord et peut faciliter le harcèlement, le doxxing ou le ciblage d'utilisateurs.

## Quelle est l'alternative ?

Utilisez la commande `/whois` de Mr. Robøt directement dans votre serveur Discord. Vous pouvez cibler un utilisateur par mention ou par identifiant.

```
/whois @utilisateur
```
