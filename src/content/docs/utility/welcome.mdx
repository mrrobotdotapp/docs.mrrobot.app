title: Souhaiter la bienvenue
description: Découvrez comment utiliser la commande /bienvenue de Mr. Robøt pour souhaiter la bienvenue aux nouveaux membres de votre serveur.
---

La commande permet de souhaiter la bienvenue à travers un message.

## Utilisation

```
/bienvenue
```

## Options
Pour le moment, aucune option n'est possible. La couleur du message est aléatoire.

## Résultat attendu

Vous recevrez un message de bienvenue de la part de Mr. Robøt.

![Résultat de la commande /bienvenue](/img/welcome/welcome.png)

## Permission

Permission par défaut : **Tout le monde peut l'exécuter**.
