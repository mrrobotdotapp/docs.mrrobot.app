La commande `/paramètres-apparence` permet de configurer les paramètres d'apparence du robot sur votre serveur Discord de Mr. Robøt.

## Utilisation

<Steps>
1. En utilisant la commande `/paramètres-apparence`, un message suivant apparaîtra
    ```
    /paramètres-apparence
    ```
    ![Message ouvert depuis la commande /paramètres-apparence sur le robot Discord Mr. Robot app](/img/appearance/appearance.png)
2. Cliquez sur le bouton de votre choix pour modifier une apparence comme la bio, l'avatar et la bannière du robot.
3. Et voilà ! Vous avez modifié les paramètres d'apparence du robot sur votre serveur Discord.
</Steps>
