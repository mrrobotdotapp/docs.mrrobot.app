La commande `/aide` est la commande d'entrée pour connaître les fonctionnalités et les autres commandes du robot.
Vous pouvez cliquer sur les commandes pour directement l'exécuter.
Vous y retrouverez également les commandes de modération, et les liens importants comme la [FAQ](/getting-started/faq/) ou le [serveur support](https://mrrobot.app/discord).

:::note
Quand vous exécutez une commande, elles sont seulement visibles par vous, et non par les autres utilisateurs.
Cela permet de ne pas polluer le salon de discussion et de garder une certaine discrétion.
:::

## Permission

Permission par défaut : **Tout le monde peut l'exécuter**.
