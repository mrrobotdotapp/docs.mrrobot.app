La commande `/serveur` permet d'afficher les informations du serveur Discord actuel. Cela inclut des détails tels que le nom du serveur, le nombre de membres, le nombre de salons, la date de création du serveur, et plus encore.

## Utilisation

```
/serveur
```

![Exemple de la commande /serveur](/img/server.png)

## Permission

Permission par défaut : **Tout le monde peut l'exécuter**.
