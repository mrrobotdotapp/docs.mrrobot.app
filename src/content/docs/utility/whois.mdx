La commande `/whois` permet de voir les informations d'un utilisateur Discord.

![Mr. Robøt : Fonctionnalité primaire, des commandes utilitaires sur Discord](/img/whois.png)

## Utilisation

Pour voir les informations d'un utilisateur, vous devez utiliser la commande `/whois` suivie de l'utilisateur ciblé.

```
/whois @utilisateur
```

![Utilisation de la commande /whois](/img/whois-result.png)

Pour voir **vos informations**, vous pouvez utiliser la commande `/whois` sans argument.

```
/whois
```

## Avec l'identifiant

Vous pouvez également utiliser l'identifiant de l'utilisateur pour voir ses informations.

```
/whois 581806230453092362
```
Cela vous rendra les mêmes informations que si vous aviez utilisé la commande `/whois @utilisateur`.

## Menu d'applications

Vous pouvez également utiliser la commande `/whois` en faisant un clic droit sur un utilisateur.

*Clic droit sur un utilisateur > `Applications` > `Plus sur son profil`.*

![Utilisation de la commande /whois avec un clic droit](/img/whois-menu.png)

## Permission

Permission par défaut : **Tout le monde peut l'exécuter**.
